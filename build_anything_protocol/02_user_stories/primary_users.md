# Primary User

## The Operator (vault owner)

The single developer who owns and uses the vault.

**Capturing**
- As the operator, I want to paste a snippet and have it saved without filling in
  a form, so capture is frictionless.
- I want the framework (React vs HTML) detected automatically, so I don't tag it.
- I want a sensible name inferred from the code, so the entry is recognizable.
- I want third-party dependencies detected, so I know what an export will need.
- I want the snippet classified into a category/subcategory, so it lands in the
  right place in the gallery.

**Previewing**
- As the operator, I want to see the component render the moment I paste it, so I
  can confirm it's the right one.
- I want that preview to be safe even if I paste something hostile or broken, so
  one bad snippet never compromises my session or the rest of the vault.

**Browsing & searching**
- As the operator, I want to browse my components grouped by the taxonomy.
- I want to search by name, tag, framework, or category and find a match fast.

**Exporting**
- As the operator, I want to multi-select components and export them as one zip.
- I want the bundle to be clean and drop-in ready for another project.

**Trusting**
- As the operator, I want the vault private to me, so nothing is reachable
  without signing in.
