# Checkpoints

Checkpoints are repo-local "save points" that capture what is true at major
milestones, per the
[Build Anything Protocol](https://github.com/Gizmo207/The-Build-Anything-Protocol).

Each checkpoint records:

- What is complete
- What is intentionally **not** started
- How the work follows the protocol
- What risks are being accepted for now
- What checkpoint comes next

## Discipline

- Each checkpoint is written once.
- If revised, add dated sections rather than rewriting history.
- Major phases do not advance without an explicit checkpoint.
